# Executable (Windows Demo Build)

This folder does **not** contain the actual Unity build output.  
The **Windows executable ZIP (~1.3 GB)** is published as a **GitHub Release asset**

Use this folder to download/unzip the Windows build quickly for evaluation.

---

## Supported platforms
- ✅ Windows 10/11

---

## What this build is

**Project:** Driving Simulator — Demo Build (Windows)  
**Engine:** Unity  (version 6000.0.29f1)
**Input device:** Logitech G29 steering wheel + pedals (recommended/required depending on your build)

This build is provided for **portfolio evaluation** only (run locally, assess features and UI flow).  
It is not intended for redistribution.

---

## Folder contents

Keep this folder small:

Executable/
README.md

After running the downloader, you will have a local folder like:

Executable/
Build/ <-- created locally after unzip
Build.zip <-- downloaded asset


> Note: `Build/` and `Build.zip` should typically be excluded from git (do not commit them).

---

## Download manually

1. Go to this repo on GitHub.
2. Open **Releases**.
3. Download the latest **Windows Build ZIP** under **Assets**.
4. Unzip it to a local folder (example):  
   `C:\Projects\DrivingSimulator\Build\`
5. Run the executable from inside the unzipped folder.
